Link Shortener

This is a simple command-line application for shortening URLs written in Java. 
It allows users to input a long URL and obtain a shortened version of that URL, 
mimicking the functionality of popular link shortening services.

Features:

Shortens long URLs to unique shortened URLs.
Handles duplicate long URLs gracefully.
Case-insensitive URL handling.
Basic error handling for invalid URL formats.
Simple command-line interface (CLI) for ease of use.

Technologies Used
Java
